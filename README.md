# odin-recipes
This project is a collection of simple recipe web pages created to practice HTML basics. It was built as a part of The Odin Project curriculum.

## About the Project
This project helped me understand:
- Basic HTML structure (<html>,<head>,<body>)
- Using headings, paragraphs, and lists
- Creating hyperlinks to navigate between different pages
- Working with local file structures

## Technologies Used
- HTML5

## How to View
You can view the project by cloning the repository and opening the (index.html) file in any web browser.
